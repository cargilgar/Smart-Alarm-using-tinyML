Detailed Description

This file contains all the constant parameters used throughout the source files.

Note
kNormalizationRanges array contains the min and max values of each feature generated while normalizing the dataset before training the model. Since our g_model expects the same format, these ranges are included as a way to process the data coming in from the sensors.
